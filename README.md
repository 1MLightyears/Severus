# Severus
A Chinese small software to combine/split a RAR archive with a picture(JPG/GIF)

这个轻量级的应用是用来制作图种的。<br>
通过这个应用，你可以<br>
\-通过将一个图片文件(JPG/GIF)和一个.rar文件头尾相接，将他们合成一个文件，同时还能正常地打开并看到图片<br>
也可以<br>
\-用WinRAR解压缩一个图种文件，并把里面的内容释放出来<br>
只需运行Severus.exe即可！
